Algorithm Visualizer

A better visualization of different algorithms made with React.

There are 6 segments

-   Pathfinder
-   Prime Numbers
-   Sorting Algorithms
-   N Queen
-   Binary Search Game
-   Binary Tree Traversals

    -   Merge sort
    -   Quick sort
-   Sieve of Eratosthenes
-   N Queen Backtracking
-   Binary Search
-   Binary Tree Traversals

    -   Preorder Traversals
